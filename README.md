A personal story site by Paul Adcock / OwlLogics. The story of being let go from Syncreon / DP World — what happened, why it happened, what was built after, and what comes next.

### Chapter One — The Story
Honest account of working at Syncreon / DP World as an engineer, delivering real value, and being shown the door. Includes a journey timeline from Before → The Day → After (every project built) → Now (GCU) → Next.

- **AutoSeq** — enterprise automotive sequencing PWA, 16 modules, 19 OEMs, SAP integration
- **OwlAI Studio** — 100% local AI platform, $0 cloud
- **RemixMii** — Suno music hub + DJ mixer, live on GitHub Pages
